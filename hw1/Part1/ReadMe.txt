IMPORTANT POINTS FOR USING USER INTERFACE FOR LINEAR EQUATION SOLVING METHODS OF PART1:
1 - JavaFX must be configuration in your ide.
2 - Write unknown number in interface , for example if there are 2 equation, write 2.
